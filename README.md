# Function Plotter（函数绘图器）

一个基于 Tkinter 的函数绘图工具，支持动态加载自定义函数并实时绘制图像。

## ✨ 功能特性

- ✅ 动态加载自定义函数
- ✅ 实时绘制函数图像
- ✅ 可调节坐标范围
- ✅ 支持多个函数同时绘制
- ✅ 清除按钮自动重置函数

---

## 🚀 使用方法

### 1. 运行主程序

### 2. 添加自定义函数
自己手动添加并且后缀改为.py


**注意事项：**
- 📌 文件名必须与函数名相同（例如：`my_sine.py` 中的函数名应为 `my_sine`）
- 📌 必须在文件开头添加 `import math`
- 📌 不保存文件将不会加载自定义代码，只会执行之前的内容

**示例：** `functions/my_sine.py`


### 3. 界面操作

- **清除按钮**：中断绘画并清除画板，同时重置函数设置
- **自动按钮**：中断并清除绘画，调用内置函数自动绘图
- **停止按钮**：停止当前绘画操作

---

## ⚙️ 坐标系说明

- 默认坐标范围：X [-10, 10]，Y [-10, 10]
- 原点位于画布中心
- 坐标轴会随画布大小自动调整

---

## 📝 自定义函数规则

| 规则 | 说明 |
|------|------|
| 文件位置 | 必须放在 `functions/` 文件夹内 |
| 文件命名 | 必须与函数名一致 |
| 导入模块 | 必须包含 `import math` |
| 函数参数 | 必须接受一个参数 `x` |
| 返回值 | 返回 `y` 值（数值类型） |

---

## 🔧 常见问题

**Q: 函数没有被加载？**  
A: 检查文件是否已保存，文件名是否与函数名一致。

**Q: 绘图出错？**  
A: 检查函数是否正确导入了 `import math`，确保返回值是数值。

**Q: 如何重置？**  
A: 点击 UI 中的清除按钮，会自动重置函数并正确设置参数。
